# A7: Discover Cryptography Used in Modern Networks

## Overview
This activity explores how cryptography is used in modern networks to protect data during transmission and ensure secure communication between systems. Cryptographic techniques help prevent unauthorized access, interception and data tampering while information is being exchanged over networks.

## Cryptographic Implementations

### 1. Wi Fi Encryption (WPA2 or WPA3)
- Secures wireless networks using encryption protocols such as WPA2 or WPA3
- Encrypts data transmitted between devices and the router
- Prevents unauthorized users from accessing the network or intercepting data
- Uses strong encryption algorithms such as AES for protection
- Security Concept: Network Security and Authentication

### 2. End to End Encryption in Messaging
- Used in messaging applications such as WhatsApp to secure communication
- Messages are encrypted on the sender's device and only decrypted on the receiver's device
- Ensures that no third party including service providers can read the messages
- Protects sensitive communication such as personal conversations and shared files
- Security Concept: Confidentiality and Data Protection

## Reflection
Cryptography plays a critical role in modern network communication by ensuring that data remains secure during transmission. Technologies such as WiFi encryption and end to end encryption help protect users from threats like data interception, eavesdropping and unauthorized access. These mechanisms are essential in maintaining trust and security in digital communication.

## Conclusion
Modern networks rely on cryptographic techniques to secure communication and protect user data. Implementations such as WPA2/WPA3 encryption and end to end encryption provide strong protection against cyber threats and ensure safe data transmission across networks.
