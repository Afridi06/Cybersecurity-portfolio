# B2: Discover 5 Unique Strong Security Implementations

## Overview
This activity explores strong security implementations which help to protect systems, data and users from cyber threats. These implementations follow best practices and significantly reduce vulnerabilities.

## Strong Security Implementations

### 1. Password Hashing with Salting
- Secure systems store passwords using hashing algorithms (e.g., bcrypt, SHA-256 with salt)
- Even if data is leaked, passwords cannot be easily reversed
- Security Strength: Data Protection and Secure Storage
