# B3: Discover 3 Proactive Security Implementations in Practice

## Overview
This activity explores proactive security implementations that are used to prevent cyber attacks before they occur. These measures focus on anticipating threats and strengthening defenses in advance.

## Proactive Security Implementations

### 1. Regular Security Patching
- Systems and applications are updated regularly to fix known vulnerabilities
- Prevents attackers from exploiting outdated software
- Example: Operating system and application updates

Proactive security measures play a crucial role in preventing cyber attacks. By addressing vulnerabilities and improving user awareness these implementations reduce the likelihood of successful attacks.
