# B1: Discover 5 Unique Weak/Vulnerable Security Implementations

## Overview
This activity explores weak or vulnerable security implementations that can expose systems, data or users to cyber threats.
Identifying such weaknesses helps in understanding common security risks and improving protection mechanisms.

## Weak Security Implementations

### 1. Storing Passwords in Plain Text
- Some systems store user passwords without encryption
- If the database is breached, attackers can directly read all passwords
- Security Risk: Data Breach and Credential Exposure
