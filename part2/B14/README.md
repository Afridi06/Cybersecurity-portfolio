# B14: Teach Your Friends About a Cybersecurity Topic

## Overview
This activity involves teaching a cybersecurity topic to others in order to improve awareness and understanding of how security systems work in real-world environments.

## Activity Conducted

- I conducted an offline teaching session with my friends in person
- The session lasted approximately 30 minutes
- The purpose of the session was to explain how intrusion detection systems (IDS) work in identifying cyber threats
- I explained the concepts using simple examples and real-world scenarios to make it easier to understand
- Security Concept: Intrusion Detection and Threat Monitoring

## Topic Taught: Intrusion Detection Systems (IDS)

### 1. What is an IDS
- Explained that an intrusion detection system monitors network or system activity
- It detects suspicious behaviour and alerts users or administrators
- Security Concept: Network Monitoring


### 2. How IDS Identifies Threats
- Explained that IDS analyses network traffic and system behaviour
- It identifies suspicious activity using predefined rules and behavioural patterns
- Security Concept: Threat Detection


### 3. Types of Detection
- Signature-based detection: compares activity with known attack patterns
- Anomaly-based detection: detects unusual behaviour different from normal activity
- Security Concept: Detection Techniques


### 4. Importance of IDS
- Helps detect cyber attacks early before damage occurs
- Provides alerts so that action can be taken quickly
- Security Concept: Early Threat Detection and Response
